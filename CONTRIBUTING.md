# Contributing to Egov.Integrations.MNotify

We welcome contributions! Here are some guidelines to help you get started.

## How to Contribute

1. **Report Bugs**: Use GitHub Issues to report bugs.
2. **Suggest Features**: Use GitHub Issues to suggest new features.
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature or bug fix.
   - Write tests for your changes.
   - Ensure all tests pass.
   - Submit a pull request.

## Development Setup

- .NET 10 SDK or later.
- An IDE like Visual Studio, Rider, or VS Code.

## Code Style

Please follow the existing code style.
